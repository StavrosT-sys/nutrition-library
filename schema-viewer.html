Easy Reference Guide for Non-Developers

What is this?

This document explains the standard format for storing nutrition information in our fitness app. Each food item in our database follows this structure to ensure consistency and completeness.

üìù Basic Information

Essential details that identify the food item.

ID text REQUIRED
A unique identifier for each food item. This helps us track and reference foods in our database.
Example: "usda_170591" or "off_5000112126619"
Name text REQUIRED
The common name of the food item.
Example: "Chicken breast, skinless, boneless" or "Coca-Cola"
Description text OPTIONAL
A more detailed description of the food item, including preparation method or specific variety.
Example: "Raw, uncooked chicken breast without skin or bones"
Brand text OPTIONAL
The brand name for packaged or commercial products.
Example: "Coca-Cola", "Kellogg's", "Dannon"
Barcode text OPTIONAL
The UPC or EAN barcode number for packaged products. Used for quick scanning.
Example: "5000112126619"

ü•Ñ Serving Size

Defines the standard portion size for the nutrition information.

Quantity number REQUIRED
The numeric amount for the serving size.
Example: 100, 250, 1
Unit text REQUIRED LIMITED OPTIONS
The unit of measurement for the serving size.
Allowed values:
g (grams) oz (ounces) ml (milliliters) cup tbsp (tablespoon) tsp (teaspoon) piece serving

üî• Calories

Calories number REQUIRED
Total energy content per serving, measured in kilocalories (kcal).
Example: 165 (for 100g chicken breast) or 105 (for 250ml Coca-Cola)

ü•ó Macronutrients (Macros)

The main nutrients that provide energy: protein, carbohydrates, and fats.

Protein number REQUIRED
Amount of protein in grams per serving. Essential for muscle building and repair.
Example: 31g (chicken breast), 0g (soda)
Carbohydrates number REQUIRED
Amount of carbohydrates in grams per serving. Primary energy source for the body.
Example: 0g (chicken breast), 26g (soda)
Fat number REQUIRED
Total amount of fat in grams per serving. Important for hormone production and nutrient absorption.
Example: 3.6g (chicken breast), 0g (soda)
Fiber number OPTIONAL
Amount of dietary fiber in grams per serving. Aids digestion and promotes gut health.
Example: 0g (chicken breast), 5g (whole wheat bread)
Sugar number OPTIONAL
Amount of sugar in grams per serving (included in total carbohydrates).
Example: 0g (chicken breast), 26g (soda)

üßà Fat Breakdown

Detailed composition of the total fat content. All fields are optional.

Saturated Fat number (g)
Fat that is solid at room temperature. Found in animal products and some plant oils. High intake may affect cholesterol levels.
Monounsaturated Fat number (g)
Healthy fat found in olive oil, avocados, and nuts. May help reduce bad cholesterol.
Polyunsaturated Fat number (g)
Healthy fat found in fish, walnuts, and flaxseeds. Includes omega-3 and omega-6 fatty acids.
Trans Fat number (g)
Artificial fat created through hydrogenation. Should be avoided as it increases heart disease risk.

üìö Source Information

Tracks where the nutrition data came from for verification and updates.

API Source text LIMITED OPTIONS
The database or API where this data originated.
Possible values:
USDA OpenFoodFacts Edamam FatSecret Nutritionix user_input
Source ID text
The unique identifier from the source database.
Example: "170591" (USDA ID)
Source URL web address
Direct link to the source data for verification.

üè∑Ô∏è Metadata

Technical information about the data record itself. All fields are optional.

Created At date/time
When this record was first added to our database.
Format: ISO 8601 (e.g., "2024-01-15T10:30:00Z")
Updated At date/time
When this record was last modified.
Verified true/false
Whether the data has been verified by a trusted source or administrator.
Confidence Score number (0-1)
A score indicating how confident we are in the accuracy of this data. 1.0 = very confident, 0.0 = uncertain.
Example: 0.95 (USDA data), 0.85 (crowdsourced data)
Notes text
Any additional notes, caveats, or important information about this data.
Example: "May vary by region" or "USDA FoodData Central - highly accurate"
